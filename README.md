# Sales Performance Dashboard | Python, SQL & Power BI

## Project Overview

This project demonstrates a complete data analysis workflow using **Python (Pandas)**, **SQL**, and **Power BI**. The goal was to clean raw sales data, analyze it using SQL queries, and create an interactive dashboard to generate meaningful business insights.

## Tools & Technologies

* Python
* Pandas
* SQL (MySQL)
* Power BI
* Excel

## Project Workflow

1. Imported and explored the raw sales dataset.
2. Cleaned the data using Pandas by handling missing values, removing duplicates, correcting data types, and formatting columns.
3. Loaded the cleaned dataset into MySQL.
4. Performed SQL analysis using SELECT, WHERE, GROUP BY, HAVING, JOIN, ORDER BY, and aggregate functions.
5. Connected SQL data to Power BI.
6. Built an interactive dashboard with KPI cards, bar charts, line charts, pie charts, slicers, and filters.

## Key Features

* Data Cleaning with Pandas
* SQL-Based Data Analysis
* Interactive Power BI Dashboard
* Business KPI Reporting
* Sales Trend Analysis
* Customer and Product Performance Analysis

## Key Insights

* Identified top-performing products and customers.
* Analyzed monthly sales trends.
* Compared sales across different categories and regions.
* Generated business insights to support data-driven decision-making.
